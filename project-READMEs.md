# 🌡️ Climate Chamber Rover

> Autonomous rover engineered for real-time climate data recording and line tracking, built at the Climate, Robotics, Rocketry & Aerospace Engineering Camp — UNC Asheville (2023).

---

## 📌 Overview

This project involved designing and programming a ground rover capable of operating inside climate test chambers. The rover autonomously followed track lines while logging environmental data, removing the need for manual monitoring inside controlled-environment chambers.

---

## ✨ Features

- ✅ Autonomous line tracking using onboard sensors
- ✅ Real-time climate data recording (temperature/environmental logging)
- ✅ Enhanced software for reliable operation in chamber conditions
- ✅ Compact hardware design suited for confined test environments

---

## 🛠️ Built With

| Layer | Tool / Technology |
|---|---|
| Programming | Python |
| Hardware | Rover chassis, line sensors, data logger |
| Software | Sensor integration scripts, data recording module |
| Environment | UNC Asheville Engineering Camp Lab |

---

## 🚀 How It Works

```
1. Rover initializes sensors on startup
2. Line tracking algorithm reads sensor array → adjusts motor output
3. Climate sensors poll at set intervals → write readings to log file
4. Data log exported for post-run analysis
```

# 🎮 Adventure Game — Blacks at Microsoft 2024

> A text-based adventure game developed in Python during the Blacks at Microsoft Summer Mentorship Program (2024). Awarded **2nd place** among all participant submissions.

---

## 📌 Overview

Built over the course of the mentorship program under guidance from Microsoft engineers, this adventure game challenges players to navigate a branching story world through decision-based gameplay. The project was a deep dive into Python fundamentals, object-oriented design, and game logic.

---

## 🎯 Gameplay

- Player navigates through a series of locations and encounters

# ⚙️ KCAL Fusion FTC Robot Code

> Autonomous and TeleOp code for KCAL Fusion's FIRST Tech Challenge (FTC) robot. Written in Java using the FTC SDK. Developed and maintained as Blue Team Captain & Lead Technician (2025–2026 season).

---

## 📌 Overview

This repository contains the software developed for KCAL Fusion's competition robot for the 2025–2026 FTC season. As Blue Team Captain and Lead Technician, I was responsible for the robot's technical systems including drive code, autonomous routines, and sensor integration.

---

## 🤖 Robot Capabilities

- **Autonomous Mode:** Pre-programmed routines for scoring during the 30-second autonomous period
- **TeleOp Mode:** Driver-controlled operation with assisted subsystem controls
- **Sensor Integration:** Encoders, distance sensors, and color sensors for feedback
- **Scouting Data:** Structured match data collection for alliance selection strategy
